# PCF Status Slider

A custom Power Apps Component Framework (PCF) control that replaces a
Choice (Option Set) field with a discrete, user-friendly status slider.

## Features
- Discrete slider with fixed steps
- Fluent UI styling
- React-based PCF control
- Field-bound (Choice / Option Set)
- Local test harness support

## Status Values
- Pending
- In-Progress
- Cancelled
- Complete

## Tech Stack
- Power Apps Component Framework (PCF)
- React
- TypeScript
- Fluent UI

## Local Development
npm install
npm start

## Deployment
pac auth create
pac pcf push --publisher-prefix demo
